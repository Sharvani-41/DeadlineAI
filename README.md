# 🚀 DeadlineAI

**AI-Powered Productivity Dashboard for Smart Task Planning**

DeadlineAI is a modern productivity dashboard that helps users organize tasks, prioritize deadlines, monitor progress, and improve daily productivity through an intuitive and responsive interface. The application focuses on making task management simple while providing AI-inspired planning and productivity insights.

---

## ✨ Features

* 📋 Create and manage daily tasks
* 🤖 AI-inspired task planning
* ⏰ Deadline tracking
* 📊 Productivity dashboard
* ✅ Mark tasks as completed
* 🗑 Delete tasks
* 💾 Persistent Local Storage
* 📈 Live progress tracking
* 💡 AI productivity insights
* 🎨 Modern responsive UI built with Tailwind CSS

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Sharvani-41/DeadlineAI.git
```

### Navigate into the project

```bash
cd DeadlineAI
```

### Install dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file:

```env
VITE_GEMINI_API_KEY=YOUR_API_KEY
```

### Run the application

```bash
npm run dev
```
